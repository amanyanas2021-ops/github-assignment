# مشروع إعادة تدوير الورق

## معلومات الطالبات
- الاسم: أماني الأمين
- الاسم: ميسم العبد الله
- المادة: إدارة المشاريع
- الجامعة: الجامعة الوطنية الخاصة
- الكلية: كلية الهندسة

## فكرة المشروع
يهدف هذا المشروع إلى إنشاء ورشة أو مركز صغير لجمع وفرز ومعالجة الورق المستعمل من المدارس والمكاتب والمنازل، وإعادة تدويره لإنتاج منتجات ورقية صديقة للبيئة.

## أهداف المشروع
- تقليل التلوث البيئي
- نشر الوعي بأهمية إعادة التدوير
- توفير فرص عمل
- دعم الاقتصاد الدائري

## النتيجة
تم توثيق مشروع إعادة تدوير الورق على منصة GitHub بما يحقق متطلبات مادة إدارة المشاريع.
